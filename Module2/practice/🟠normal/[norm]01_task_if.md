## "Трехзначное ли"

### Задание

Проверьте, является ли данное число трехзначным.

### Формат входных данных

Дано целое число.

### Формат выходных данных

Вывести "Да", если данное число трехзначное и "Нет" в противоположном случае.

### мое решение


n = int(input())
if n < 999 and n > 0:
    print('Yes')
else:
    print('No')
